## 1. Начало работы

Система KMS работает в браузере. Установка не требуется.

**Адрес приложения:** предоставляется администратором системы (URL Streamlit Cloud).

**Поддерживаемые браузеры:** Chrome, Firefox, Safari, Edge (актуальные версии).

**Мобильные устройства:** поддерживаются.

---

## 2. Вход в систему

### Первый запуск (нет суперадмина)
1. Откройте приложение в браузере
2. На странице входа будут видны две вкладки: **Вход** и **Регистрация**
3. Перейдите на вкладку **Регистрация**
4. Введите email и пароль (минимум 6 символов)
5. Нажмите **Создать суперадмина**
6. После создания войдите через вкладку **Вход**

### Обычный вход
1. Откройте приложение
2. Введите email и пароль
3. Нажмите **Войти**

> **Примечание:** После создания суперадмина вкладка «Регистрация» исчезает — новые пользователи добавляются только суперадмином.

---

## 3. Настройки

### Язык интерфейса
- До входа: выберите язык в выпадающем меню в верхнем левом углу страницы входа
- После входа: в левом боковом меню раздел **Настройки** → выбрать язык → нажать **Сохранить**

### Валюта
- Аналогично языку — выбирается до или после входа
- Доступные валюты: **Шекели (₪)**, **Доллар ($)**, **Евро (€)**
- Все суммы в системе отображаются в выбранной валюте

> Настройки сохраняются в вашем профиле и будут восстановлены при следующем входе.

## 4. Суперадмин: управление садиками

После входа суперадмин видит **список садиков**.

### Просмотр садиков
- Каждый садик отображается карточкой с названием, адресом и телефоном
- Кнопка **Войти →** открывает данный садик

### Вход в садик
1. Нажмите **Войти →** на карточке садика
2. Откроется главная страница садика с полным меню
3. Для возврата к списку садиков нажмите **← Назад** в верхней части бокового меню

### Добавление садика
1. В боковом меню выберите **Детские сады (⚙️)**
2. Перейдите на вкладку **Добавить садик**
3. Заполните: Название (обязательно), Адрес, Телефон, URL логотипа
4. Нажмите **Добавить**

### Редактирование садика
1. В **Детские сады (⚙️)** → вкладка **Список садиков**
2. Раскройте нужный садик
3. Заполните новые данные в форме редактирования
4. Нажмите **Сохранить**

### Удаление садика
1. В **Детские сады (⚙️)** → найдите садик
2. Нажмите кнопку **🗑 Удалить «Название»**

> **Внимание:** Удаление садика удаляет ВСЕ связанные данные (дети, посещаемость, продукты, расходы, платежи). Действие необратимо.

### Инициализация базы данных (сброс)
В разделе **Детские сады (⚙️)** есть скрытый блок **🛠 Инициализация базы данных**:
- Полностью сбрасывает базу данных
- Создаёт тестовые данные (3 садика с детьми, посещаемостью, продуктами, расходами, платежами)
- Используется только при первоначальной настройке или тестировании

## 5. Суперадмин: управление администраторами

В боковом меню: **👤 Администраторы**

### Добавление администратора
1. Вкладка **Добавить администратора**
2. Email, пароль, выбор садика
3. Нажмите **Создать**

### Просмотр администраторов
- Список всех администраторов с их садиком
- Поддерживается смена пароля и удаление

> Администратор видит только свой садик и не имеет доступа к разделу администраторов.

## 6. Главная страница (дашборд)

После входа в садик отображается главная страница с:

### Предупреждения
- **Должники:** если есть дети с неполной оплатой за текущий месяц — жёлтое предупреждение с количеством должников
- **Низкий остаток:** если какой-либо продукт ниже минимального остатка — жёлтое предупреждение со списком продуктов

### Навигация
Боковое меню содержит разделы:
- 🏠 Главная
- 👦 Дети
- 📅 Посещаемость
- 🍎 Продукты
- 💰 Расходы
- 📊 Отчёты
- 💳 Оплата
- 👤 Администраторы (только суперадмин)

---

## 7. Дети

Раздел **👦 Дети**

### Просмотр детей
- Список детей текущего садика
- Фильтры: группа (все / младшая / средняя / старшая), статус (активный / неактивный)
- Отображается: ФИО, возраст, группа, контакт родителя, ежемесячный взнос

### Добавление ребёнка
1. Вкладка **Добавить ребёнка**
2. Заполните: Имя, Фамилия (обязательно), Дата рождения, Имя родителя, Телефон родителя, Дата зачисления, Группа, Ежемесячный взнос, Статус
3. Нажмите **Добавить**

### Редактирование ребёнка
1. В списке детей раскройте карточку ребёнка
2. Измените нужные данные
3. Нажмите **Сохранить**

### Деактивация ребёнка
- Измените статус на **Неактивный** при редактировании
- Неактивные дети не показываются в посещаемости и не учитываются как должники

## 8. Посещаемость

Раздел **📅 Посещаемость**

### Просмотр журнала
- Выберите дату (по умолчанию — сегодня)
- Фильтр по группе (опционально)
- Таблица показывает всех активных детей и их статус за выбранный день

### Отметка посещаемости
1. Выберите дату
2. Для каждого ребёнка выберите статус:
   - **Присутствовал** — ребёнок был в садике
   - **Отсутствовал** — пропустил без причины
   - **Болел** — пропустил по болезни
3. Нажмите **Сохранить посещаемость**

### Итоги
Под таблицей отображается сводка:
- Количество присутствующих
- Количество отсутствующих
- Количество болеющих

## 9. Склад продуктов

Раздел **🍎 Продукты**

### Просмотр склада
- Список продуктов с текущим остатком
- Красная пометка — остаток ниже минимального уровня

### Добавление продукта
1. Вкладка **Добавить продукт**
2. Название, единица измерения (кг, литры, штуки и др.), минимальный остаток
3. Нажмите **Добавить**

### Приход / Расход
1. Выберите продукт из списка
2. Вкладка **Приход** или **Расход**
3. Введите количество и дату
4. Нажмите **Добавить транзакцию**

### История транзакций
- Выберите продукт — отображается история всех приходов и расходов

---

## 10. Расходы

Раздел **💰 Расходы**

### Просмотр расходов
- Фильтр по месяцу и категории
- Итого за период

### Добавление расхода
1. Вкладка **Добавить расход**
2. Дата, категория, сумма, описание (опционально), комментарий (опционально)
3. Нажмите **Добавить**

**Категории:**
- Еда
- Транспорт
- Жилье
- Связь
- Другое

## 11. Оплата родителей

Раздел **💳 Оплата**

### Добавление платежа
1. Вкладка **Добавить платёж**
2. Выберите ребёнка, год, месяц
3. Введите сумму и дату оплаты
4. Нажмите **Добавить**

> Можно добавить несколько платежей для одного ребёнка за один месяц (частичные оплаты). Система автоматически суммирует их при проверке задолженности.

### Список должников
1. Вкладка **Должники**
2. Выберите год и месяц
3. Отображается список детей с суммой долга (взнос − оплачено)

**Ребёнок считается должником если:**
- Суперадмин ежемесячный взнос > 0
- Сумма всех платежей за месяц < ежемесячного взноса

### История платежей
1. Вкладка **История**
2. Выберите ребёнка — отображаются все его платежи

---

## 12. Отчёты

Раздел **📊 Отчёты**

### Отчёт по посещаемости
1. Вкладка **Посещаемость**
2. Выберите год и месяц
3. Отображается таблица: ребёнок × дни месяца
4. Кнопка **📥 Скачать Excel** — экспорт в файл

### Финансовый отчёт
1. Вкладка **Финансы**
2. Выберите год и месяц
3. Отображается:
   - Итого поступлений (сумма платежей)
   - Итого расходов (по категориям)
   - Баланс (доходы − расходы)
4. Кнопка **📥 Скачать Excel** — экспорт в файл

---

## 13. Выход из системы

В нижней части бокового меню нажмите кнопку **Выйти**.

Сессия завершается, данные сохраняются в базе данных.

## Часто задаваемые вопросы

**Q: Я ввёл оплату, но ребёнок всё равно числится должником.**  
A: Проверьте, что сумма всех платежей за месяц равна или превышает ежемесячный взнос ребёнка. Частичные платежи суммируются автоматически.

**Q: Я не вижу детей другого садика.**  
A: Это нормально. Каждый администратор видит только детей своего садика.

**Q: Как добавить нового администратора?**  
A: Только суперадмин может добавлять администраторов через раздел **👤 Администраторы**.

**Q: Как изменить ежемесячный взнос ребёнка?**  
A: Раздел **👦 Дети** → найдите ребёнка → отредактируйте поле «Ежемесячный взнос» → Сохранить.

**Q: Можно ли восстановить удалённые данные?**  
A: Нет. Удаление необратимо. Будьте осторожны при удалении садиков, детей и данных.

**Q: Как экспортировать данные?**  
A: Раздел **📊 Отчёты** → выберите тип отчёта → кнопка **📥 Скачать Excel**.
